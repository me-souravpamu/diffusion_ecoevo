This repository contains the codes for the reproduction of the figures for the paper entitled "Diffusion-induced instabilities promote cooperation in eco-evolutionary networks" by Sourav Roy, Md Sayeed Anwar, Timoteo Carletti, Matjaz Perc, and Dibakar Ghosh. 

Sourav_pattern_network.ipynb: Pattern formation (Fig. 2),
pattern_amplitude.ipynb: Amplitude vs kappa (Fig. 3).
max_eigen_single.m: Dispersion relation
